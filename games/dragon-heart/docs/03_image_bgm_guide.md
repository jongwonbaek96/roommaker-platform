## 4. 힌트 = "불씨" (연출된 자원)

힌트는 버튼이 아니라 **소모품**이다.

- 표기: `🕯️ ×N`
- 사용 시: *"힌트의 불씨를 하나 태웁니다."* → 확인 → 하나 감소
- 다 떨어지면: *"힌트의 불씨가 모두 꺼졌다…"*
- 엔딩에 **남은 불씨**가 기록된다.

**어둠 속에서 불을 태워 길을 본다** — 이 게임의 주제(빼앗은 심장 / 되돌리는 선택)와 정확히 맞물리는 장치다. 건드리지 말 것.

## 5. 각인 획득 연출

```

```

- 획득 시 0.6초 지연 후 토스트(`good`).
- 상단 바에 `각인 ◆◆◇◇` 형태로 진행도가 표시된다(`fragBar()`).
- **숫자를 그 자리에서 알려주지만, 적어두지 않으면 30번 미션에서 곤란해진다.** 의도된 설계.
