# 6. 온라인 구현 참고 (룸메이커 플랫폼)

- **입력 유형 분포**: 숫자 자물쇠 7개(1,4,6,13,17 등) / 텍스트 입력 5개(9,11,16,19,20) / 배열·조작 인터랙션 8개(2,5,8,10,12,14,15,18)
- **핵심 인터랙션**: 12번 암전+별 잇기(드래그), 10번 저울(드래그 앤 드롭), 20번 문 당기기(버튼을 '밀기/당기기' 두 개로 제공)
- **회수 요소(복선) 체크리스트**: ① 7:16 (1번→17번) ② 밀기/당기기 (프롤로그→20번) ③ 문틀 글자 5개 (각 장→19번) ④ 휴대폰 (2번→13번) ⑤ 벤치 (14번 편지의 공간 일치)
- **엔딩 후 화면**: 플레이어가 20번에서 입력한 '나에게 남긴 한마디'를 카드 이미지로 저장/공유 가능하게 하면 바이럴 요소가 된다.
