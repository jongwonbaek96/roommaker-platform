## 2. 팔레트 (실측 CSS 변수)

| 변수 | 값 | 용도 |
|---|---|---|
| `--bg` | `#0a0505` | 정전된 방송국 (거의 검정, 아주 옅은 적색기) |
| `--panel` | `#150c0a` | 패널 |
| `--panel2` | `#1d1210` | 모달 |
| `--border` | `#3a2420` | 테두리 |
| `--dim` | `#7a6a5a` | 보조 텍스트 |
| `--gold` | `#c9a84c` | 아날로그 계기 · 파형 (진공관의 색) |
| `--green` | `#7fb069` | 정답 |
| `--accent` | `#ff3b30` | **ON AIR 램프** (registry accent) |
| `--danger` | `#ff3355` | 경고 알림 |

**색 설계 의도**: 화면 전체가 어두운 적갈색 — *꺼진 ON AIR 램프의 잔광*. 유일하게 선명한 것은 계기의 금색(`--gold`)과 온에어 빨강(`--accent`).

## 3. 연출 원칙 — "무섭게 하지 말고, 이상하게 해라"

이 게임의 공포는 점프스케어가 아니라 **어긋난 디테일**이다.

| 장치 | 예시 |
|---|---|
| **알림(`showNotif`) 지연 연출** | 정답 → 0.7초 뒤 무해한 문구 → **2.6초 뒤 서늘한 문구** (`g2`: 테이프 재생 후 *"복도에서 젖은 발소리. 문틈으로 보이는 복도에는 — 아무도 없다."*) |
| **오답 메시지가 곧 복선** | `q5` 에 `0000`/`2400` 입력 → *"…시계는 자정에 닿지 못했다."* |
| **기계가 혼자 움직인다** | `q3` 타자기, `q4` 프린터 — 사람은 끝까지 등장하지 않는다 |
| **danger 색 알림** | 진실에 가까운 문장에만 `'danger'` 플래그를 쓴다 |

## 4. BGM · 사운드 — **없음** (실측 0건)

4. **ON AIR 진입(`g8`)** — 무음 → 갑자기 방송 시그널.
