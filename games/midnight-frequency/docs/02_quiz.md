## GAME 02 — 끊어진 릴 테이프  `g2`

**과제**: 잘린 테이프 조각을 **파형이 이어지도록** 원래 순서(0→6)로 재배열

**✅ 정답 / 클리어 조건**

**클리어 후**: 서하율 최종회 육성 재생: *"…저는 조금 더, 여기 남기로 했어요. 기다리는 사람이 있거든요. 언젠가, 꼭 올 사람이."* → 직후 **복도에서 젖은 발소리. 그런데 복도엔 아무도 없다.**
