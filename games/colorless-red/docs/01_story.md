## 0. 시리즈 절대 규칙 (v4 톤 3원칙)

1. **마지막까지 냄새 금지** — 엔딩 전에는 각 색깔의 문제를 암시하는 단어·연출·오답 메시지·힌트가 **단 하나도** 등장하지 않는다. 중간 리빌(일기 다시 읽기 진실 공개, 섬뜩한 잉크, 폴라로이드 진실 컷) → **전부 폐기.** 장 클리어 연출은 "일기가 예쁘게 물들었어!"로.
2. **아이의 언어** — 모든 문장은 아이가 쓴 그림일기 문체. 미션 이름·힌트·오답 반응까지. (오답: "들켰다!" ❌ → "찾았대! 헤헤, 다시 숨자!" ⭕)
3. **복선은 '자연스러운 디테일'** — 플레이어가 "귀엽네" 하고 지나칠 수 있어야 진짜 복선이다. 게임은 절대 아는 척하지 않는다.

## 3. 엔딩 리빌 시퀀스 (전 에피소드 공통 구조)

```
30번 미션 클리어 → 캐릭터 100% 채색
→ "일기를 처음부터 다시 읽어 볼까?"
→ [복선 몽타주] 각 장의 삽화가 컬러로 하나씩, 자막이 아이의 문장 → 어른의 문장으로 (장당 1컷, 총 5컷)
→ [리빌 이미지] 신규 이미지 1장 — 진실을 담은 단 한 컷 (직접 묘사 없이 상징으로)
→ [진실 문장] 한 줄
→ PRISM 조각 지급 → "그리고 문이 열렸다" → 다음 세계로
```

### 🔴 빨강의 리빌

몽타주 5컷 → 리빌 이미지(붉은 네온이 켜진 밤거리, 그 앞에 선 그녀의 뒷모습 — 이번엔 전부 컬러) → 진실 문장: *"그녀가 사람들을 웃게 하던 곳은, 어른들의 밤이었다."* → *"그래도 — 이번엔 그녀가 고른 문이다."* → 문.

**게임 내 실측 엔딩 텍스트**
- `endText`: 문틈으로 새어드는 빛에, 립스틱만 빨갛게 물든다. 그녀는 뒤돌아보지 않는다.
- `credit`: 그녀의 색은, 도망친 곳에서 시작되었다.
- `reveal` 데이터: **있음** (v4 몽타주 시스템 탑재)
