## 1. 채색 시스템 (시리즈 핵심 연출)

세상은 **흑백**이다. 문제를 풀 때마다 진행도가 오르고, 그에 비례해 **캐릭터 아바타(SVG)와 UI가 점점 색으로 물든다.**
30번을 풀면 **채색 100%** → 엔딩 리빌 몽타주 → PRISM 조각.

- 구현: `colorless-core*.js` 의 채색 로직 + `colorless-style.css`
- **한 곳을 고치면 5개 에피소드 전부 바뀐다.** (공유 엔진)

### 🚨 실측 경고 — 이미지 파일이 **하나도 존재하지 않는다**

`games/assets/red/` 폴더는 **비어 있다.** 위 8장 전부 404다.
게임은 실행되지만 그림일기 삽화와 미션 배경이 전혀 뜨지 않는다.

**해야 할 일**: 8장을 생성해 `games/assets/red/` 에 넣는다. (파일명 그대로)

| 파일 | 생성 프롬프트 방향 (v4 톤 — 아이의 그림일기, 해맑음 100%) |
|---|---|
| `d1.webp` | 1장 "달님" — 크레파스 그림체. 밤하늘의 커다란 달, 그 아래 예쁘게 차려입고 걸어가는 언니의 뒷모습. **밝고 귀엽게.** 흑백 기조 + 아주 옅은 붉은 기운만. |
| `d2.webp` | 2장 "웃음" — 웃고 있는 손님들(얼굴은 단순한 스마일). 아이가 그린 "사람들을 웃게 하는 일". |
| `d3.webp` | 3장 "씩씩" — 알약과 따뜻한 차 한 잔, 이불 속에서 씩씩하게 브이. |
| `d4.webp` | 4장 "숨바꼭질" — 골목 담벼락 뒤에 꼭꼭 숨은 아이 캐릭터, 즐거운 표정. |
| `d5.webp` | 5장 "두근두근" — 새벽 기차, 커다란 가방, 설레는 표정. |
| `s_vanity.webp` | R-01 미션 배경 — 화장대(립스틱·거울·메모지). 크레파스 톤. |
| `s_station.webp` | 5장 미션 배경 — 새벽 기차역 플랫폼. |
| `s_door.webp` | 엔딩 — 빛이 새어드는 문. |

**리빌 이미지(v4 §3)**: 몽타주 마지막에 들어갈 **신규 1장** — *붉은 네온이 켜진 밤거리, 그 앞에 선 그녀의 뒷모습. 이번엔 전부 컬러.* (아직 미생성)

> 이미지 톤 절대 규칙: **엔딩 전 이미지에는 진실의 냄새가 단 하나도 없어야 한다.** 어둡고 서늘한 연출 금지.

## 5. BGM · 사운드

**없음.** 5개 에피소드 전부 오디오를 쓰지 않는다 (실측 0건).
넣는다면 공용 엔진 `colorless-core*.js` 에 한 번만 추가하면 5편 전부 반영된다.

- 권장: 흑백 구간엔 뮤트/저역만, **채색이 진행될수록 악기가 하나씩 들어오는** 구조. (채색 시스템과 같은 축)
- 엔딩 리빌: 몽타주 5컷 동안 음이 쌓이다 진실 문장에서 **뚝 끊기는** 연출.
