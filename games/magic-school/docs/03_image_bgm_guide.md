## 2. 팔레트 (실측)

| 변수 | 값 |
|---|---|
| `--purple-dark` | `#0d0520` (배경 — registry bg와 동일) |

| `--purple-light` | `#4a2080` |
| `--purple-bright` | `#9b59b6` |
| `--purple-ghost` | `#e8d5ff` |
| `--gold` | `#ffd700` (registry accent) |
| `--gold2` | `#ffb300` |
| `--blue` | `#2980b9` · `--green` `#27ae60` | 4원소 마법진(P2) 버튼 |

**⚠️ 색이 곧 정답인 퍼즐이 있다.** P2(4원소 마법진: 빨강→파랑→초록→노랑)와 P13(사이먼)은 **버튼 색으로 정답을 표현**한다. 팔레트를 바꾸면 힌트 문구("빨간색→파란색…")와 어긋난다. **색과 힌트를 세트로 수정할 것.**

## 4. 연출

- **교내 방송(인터콤)** 이 유일한 "연출 장치"다. 순환 재생되며 코미디를 담당한다. → `01_story.md` §2
- 이미지도 사운드도 없으므로, **톤은 전적으로 텍스트에 실려 있다.** 문구를 고치면 게임의 성격이 바뀐다.

## 5. BGM · 사운드 — **없음**

인터콤 대사가 *"(게임 효과음)"* 이라고 **글자로** 적혀 있다(인터콤 3번). 소리로 바꾸면 개그가 훨씬 산다.

넣는다면:
- 교내 방송 시작 **딩동댕** 차임 — 인터콤 등장마다.
- 교장이 하는 모바일 게임 효과음 — 인터콤 3번에서 실제로 들리게.
- 마법탑 BGM — 웅장하게 깔았다가, **P15에서 비상구 스티커를 찾는 순간 뚝 끊기기.** (이 게임 최고의 개그 타이밍)
