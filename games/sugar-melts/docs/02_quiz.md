# 챕터 1 — 방명록 (응접실)

## S01 [미션] 방명록 서명 + 봉인 문장(紋章) 회전 정렬 — 난이도 1
- **방**: 응접실 | **메커니즘 유형**: 회전 정렬 + 텍스트 입력
- **메커니즘 상세**: ① 방명록에 이름 입력(텍스트 필드, 저장 → E2 엔딩 회수). ② 기록국 초대장의 사탕 봉인 = 동심원 3겹 원판(바깥: 하얀 새 / 중간: 오븐 / 안쪽: 과자집). 원판을 탭할 때마다 45°씩 회전. 3겹의 문양이 벽난로 위 문장(紋章)과 같은 자세로 정렬되면 봉인이 갈라지며 인증 체크리스트 수령.
- **UI 스케치**: 화면 중앙 원형 봉인(3겹 링), 각 링 탭 영역 구분. 우상단에 벽난로 문장 참조 버튼(확대 보기). 하단 방명록 서명란.
- **정답/클리어**: 새 부리 ↑ / 오븐 문 ↓ / 굴뚝 ↖ 정렬 (각 링 8방향 중 1위치).
- **힌트1**: 이 봉인, 어디서 본 그림 아닌가? 이 방 안에 똑같은 문장이 있다.
- **힌트2**: 벽난로 위 문장을 확대해서 세 문양의 방향을 그대로 따라 하라.
- **힌트3**: 바깥 링은 새 부리가 위, 중간 링은 오븐 문이 아래, 안쪽 링은 굴뚝이 왼쪽 위.
- **정답 공개**: "새↑·오븐↓·굴뚝↖. 이 집의 문장은 65년간 한 번도 바뀐 적이 없다 — 그래서 열쇠가 된다."
- **오답 처리**: 무제한 시도. 2분 경과 시 벽난로 문장이 화면 구석에서 반짝임.
- **아이템**: 획득 — 인증 체크리스트(전시물 12점 정본 목록 — S28 목록과 동일: 조약돌 주머니 포함, 레시피 카드 제외). 서명 데이터 저장(E2).

## S02 [퀴즈] 동화 삽화 6점 순서 배열 — 난이도 1
- **방**: 응접실 | **메커니즘 유형**: 탭 순서 배열 (이 UI가 S31에서 재사용됨 — 인수인계 메모 3번)
- **메커니즘 상세**: 벽의 삽화 6점이 카드로 표시. 탭한 순서대로 ①~⑥ 번호가 붙고, 재탭하면 취소. 6장 모두 지정 후 "확정" 버튼.
- **UI 스케치**: 2×3 카드 그리드, 카드 좌상단에 순서 뱃지. 하단 확정 버튼. (S31과 완전 동일 레이아웃으로 구현할 것)
- **정답/클리어**: 조약돌 줍는 헨젤 → 빵부스러기 먹는 새 → 과자집 발견 → 우리 속 헨젤 → 오븐 앞 그레텔 → 보석 안고 귀향.
- **힌트1**: 다들 아는 그 동화의 순서 그대로다. 기억을 믿어라.
- **힌트2**: 시작은 집을 떠나기 전 밤(조약돌), 끝은 할미가 제일 좋아한다는 "돌아가는 그림".
- **힌트3**: 조약돌 → 새 → 과자집 → 우리 → 오븐 → 귀향.
- **정답 공개**: "조약돌→새→과자집→우리→오븐→귀향. 모두가 아는 순서 — 지금은. (이 배열 기준선이 훗날 다시 쓰인다)"
- **오답 처리**: 확정 시 틀린 장수만 표시("2장이 제자리가 아니다"), 위치는 비공개.
- **아이템**: 배열 데이터 → S31 대칭 연출 기준선으로 저장.

## S03 [미션] 벽난로 사탕 문장 다이얼 → 진열장 개방 — 난이도 2
- **방**: 응접실 | **메커니즘 유형**: 3자리 심볼 다이얼 (관찰 단서 결합)
- **메커니즘 상세**: 벽난로 옆 진열장에 숫자 다이얼 3개(1~6). 삽화 액자 중 하나의 뒷면(액자 탭→뒤집기)에 각인: *"새를 따라가라. 새가 앉은 그림의 번호를."* → S02에서 배열한 삽화 중 하얀 새가 그려진 그림은 2·3·6번(빵부스러기 장면, 과자집 발견 장면의 지붕, 귀향 장면의 어깨 위).
- **UI 스케치**: 진열장 클로즈업 + 다이얼 3개(상하 화살표 탭). 삽화는 각 카드 길게 누르면 뒤집힘.
- **정답/클리어**: 2-3-6 입력 → 진열장 개방.
- **힌트1**: 진열장은 숫자를 원한다. 이 방에서 번호가 붙은 것은 삽화뿐이다.
- **힌트2**: 액자 하나의 뒷면에 글귀가 있다. 전부 뒤집어 보라.
- **힌트3**: 하얀 새가 등장하는 삽화는 두 번째, 세 번째, 여섯 번째다.
- **정답 공개**: "2-3-6. 새가 있는 그림의 번호 — 이 집에서는 언제나 새를 따라가면 길이 열린다."
- **오답 처리**: 3회 오답 시 첫째 다이얼이 정답(2)에 고정.
- **아이템**: 획득 — **하얀 조약돌 주머니**(확대 관찰 시 안에서 바느질 골무 발견 — 무해 디테일, 언급 없이 수첩에만 기록. F06).

# 챕터 2 — 마녀의 오븐 (과자 부엌)

## S04 [미션] 찢어진 레시피 카드 복원 — 난이도 2
- **방**: 과자 부엌 | **메커니즘 유형**: 조각 스왑 퍼즐
- **메커니즘 상세**: 진저브레드 레시피 카드가 12조각으로 찢겨 개수대·선반·밀가루 통에 흩어짐(핫스팟 탐색으로 12조각 수집 → 복원대에서 3×4 그리드). 조각 2개를 차례로 탭하면 자리 교환. 그림 결(생강 뿌리 일러스트)과 글줄이 단서.
- **UI 스케치**: 상단 완성 실루엣 희미하게, 중앙 3×4 그리드, 조각 탭 시 테두리 발광.
- **정답/클리어**: 12조각 전부 정위치 → 카드 뒷면 자동 공개: *"장작은 마른 것부터 셋. 풀무는 세 번, 쉬고, 두 번."*
- **힌트1**: 모서리 조각(테두리 선이 있는 것) 4개부터 자리를 잡아라.
- **힌트2**: 배합비 숫자는 위에서 아래로 커진다. 밀가루 3 → 꿀 6 → 생강 9.
- **힌트3**: 남은 두 조각이 헷갈리면 — 생강 뿌리 일러스트의 잔뿌리가 오른쪽 아래로 흐른다.
- **정답 공개**: "배합비 카드 완성. 뒷면의 점화 메모가 진짜 보상이다: '장작 셋, 마른 것부터. 풀무 세 번, 쉬고, 두 번.'"
- **오답 처리**: 벌점 없음. 90초 경과마다 정위치 조각 1개 자동 고정.
- **아이템**: 획득 — **점화 메모** (→S05 사용).

## S05 [미션] 오븐 점화 — 장작 배열 + 풀무 타이밍 — 난이도 2
- **방**: 과자 부엌 | **메커니즘 유형**: 순서 배치 + 타이밍 탭 (T1 발동)
- **메커니즘 상세**: ① 장작 6개(잔가지·중간·통장작 × 마른 것/젖은 것) 중 마른 것 3개를 가는 것부터 순서대로 아궁이에 탭 배치. ② 불씨 게이지: 바늘이 좌우로 흔들리는 원형 게이지, 초록 구간에서 풀무 버튼 탭. 점화 메모의 리듬대로 "3연타 — 한 박 쉼 — 2연타" 성공 시 게이지 100%.
- **UI 스케치**: 아궁이 단면 + 장작 선택 트레이. 점화 후 화면 전환: 대형 원형 게이지 + 풀무 버튼 1개.
- **정답/클리어**: 마른 잔가지→마른 중간→마른 통장작 배치 + 풀무 리듬 성공.
- **힌트1**: 젖은 장작은 표면이 검다. 불은 가는 것부터 먹는다.
- **힌트2**: 레시피 카드 뒷면 메모가 풀무의 박자다.
- **힌트3**: 초록 구간에서만 탭. 탭-탭-탭, 쉬고, 탭-탭.
- **정답 공개**: "마른 장작 세 개를 가는 순서로, 풀무는 3·쉼·2. 불이 오르자 — 위층 어딘가에서 물방울 떨어지는 소리가 들린다."
- **오답 처리**: 불씨 꺼짐 → 즉시 재시도. 3회 실패 시 초록 구간 폭 1.5배 확대.
- **아이템**: 소모 — 점화 메모(수첩 보관). **T1 발동**: 다락 채광창 성에가 녹기 시작(연출: "다락 쪽에서 똑, 똑" 자막).

## S06 [퀴즈] 안내판 치수 검산 — 난이도 3
- **방**: 과자 부엌 | **메커니즘 유형**: 측정 + 숫자 입력 (불일치 로그 #1)
- **메커니즘 상세**: 오븐 안내판: *"아궁이 폭 3자(尺) — 어른이 걸어 들어갈 만큼 넓다."* 개수대 서랍에서 접이식 줄자 획득 → 아궁이 좌우 끝을 차례로 탭하면 실측치 표시: **1자 8치**. 문제: "안내판의 수치와 실측의 차이는 몇 치인가?" (안내판 각주: 1자 = 10치)
- **UI 스케치**: 안내판/실측 화면 분할. 하단 숫자 키패드.
- **정답/클리어**: **12** (치) 입력.
- **힌트1**: 안내판을 믿지 말고 네 손의 줄자를 믿어라.
- **힌트2**: 3자를 치로 바꿔라. 각주에 환산이 적혀 있다.
- **힌트3**: 30치 − 18치.
- **정답 공개**: "12치. 안내판의 오븐은 실물보다 한 뼘 반이나 넓다. 어른이 '걸어 들어갈' 수 있는 건 안내판 속 오븐뿐이다. (할미: '아이고, 이건 복제품이야.')"
- **오답 처리**: 2회 오답 시 줄자 눈금 클로즈업 + 환산 각주 하이라이트.
- **아이템**: **불일치 로그 #1 기록**(치수 모순 + 그을음이 바닥 틈새 아래쪽에 몰린 사진 — 그을음은 관찰 핫스팟으로 자동 첨부. F02).

## S07 [미션] 덤웨이터 도르래 개통 — 로프 경로 + 균형추 — 난이도 2
- **방**: 과자 부엌 | **메커니즘 유형**: 경로 선택 퍼즐 (T4 발동)
- **메커니즘 상세**: 벽감 속 소형 승강기. 후크 6개 중 4개를 순서대로 탭해 로프 경로를 지정(로프가 교차하면 실패 — 교차 없는 유일 경로 1개). 마지막으로 균형추 3종(모래주머니 小·中·大) 중 선택 — 승강기 접시에 새겨진 정원(定員): *"빵 두 덩이"* → 저울 그림 옆 각주 *"빵 한 덩이 = 표준 주머니 반. 표준은 가운데 크기."* + 中 주머니 마대에만 **"한 주머니(표준)" 각인** → 中 확정.
- **UI 스케치**: 벽 단면도에 후크 6개(점), 탭 순서대로 로프 선이 그려짐. 하단 추 3개 선택 버튼.
- **정답/클리어**: 무교차 경로 + 中 추 → 덤웨이터 개통.
- **힌트1**: 로프는 스스로와 만나면 안 된다. 선을 머릿속으로 먼저 그려 보라.
- **힌트2**: 위쪽 두 후크는 반드시 연속으로 지나야 한다. 추가 고민이면 접시의 정원(定員)과 저울 각주를 읽어라.
- **힌트3**: 추는 '빵 두 덩이 = 표준 한 주머니'. "한 주머니(표준)" 각인이 있는 가운데 것이다.
- **정답 공개**: "교차 없는 외길 + 중간 추. 음식을 내리던 길이 이제 도구를 나른다."
- **오답 처리**: 추가 틀리면 승강기가 쿵 내려앉는 연출 후 리셋(벌점 없음).
- **아이템**: **T4 발동** — 부엌↔저장고 전달 통로 개통. 개통된 승강기 안에서 **녹슨 크랭크 손잡이** 발견(저장고 쪽 문으로 회수 → S23 사용).

# 챕터 3 — 숫자가 맞지 않는다 (식료품 저장고)

## S08 [미션] 대저울 균형 맞추기 — 난이도 3
- **방**: 저장고 | **메커니즘 유형**: 분동 조합 (양팔저울)
- **메커니즘 상세**: 선반의 봉인 상자(무게 11)를 저울 왼쪽 접시에 올리라는 각인: *"들어온 만큼만 열린다."* 분동은 1·3·9 세 개뿐. 분동을 탭 → 좌/우 접시 선택으로 배치(양쪽 배치 가능). 평형이 되면 선반 뒤 벽판이 밀려 열림.
- **UI 스케치**: 중앙 대저울(기울기 실시간 애니메이션), 하단 분동 트레이, 접시 탭으로 회수.
- **정답/클리어**: 오른쪽 9+3, 왼쪽(상자 쪽) 1 → 평형(11+1 = 12 = 9+3).

## S09 [퀴즈] 장부 기호 해독 — 출납 역전 — 난이도 3

- **UI 스케치**: 피리 가로 뷰 + 구멍 토글 3개 + 불기 버튼(누르는 길이 = 음 길이). 상단에 수첩 기록 2종 소환 버튼.
- **정답/클리어**: 음높이 1·6·4 + 리듬 장-단-장으로 연주 → 뒷방 개방.
- **힌트1**: 자물쇠에 열쇠 구멍이 없다 — 듣는 자물쇠다. 재료는 이미 수첩에 두 가지가 있다. 규칙 카드의 문장을 끝까지 읽어라.

- **힌트3**: 창살 1·6·4에 해당하는 운지로, 길게-짧게-길게.
- **정답 공개**: "1·6·4, 장-단-장. 새를 부르던 세 음이 문을 연다. 이 집의 마지막 방 — 설탕이 한 겹도 없는 방으로."
- **오답 처리**: 오연주 시 자물쇠 무반응(정적). 3회 실패 시 자물쇠 옆 운지 대응표 새김 하이라이트. 5회 실패 시 리듬 가이드 점멸 재생.
- **아이템**: 사용 — 뼈 피리(보유 유지), 쪼기 순서·오르골 기록. **뒷방 개방** → 유품 상자·그레텔의 편지 이벤트(Ch9 진입).

## S30 [미션] 유품 순례 — 세 물건을 그날 밤 자리로 — 난이도 4
- **방**: 전 방(저장고·온실·우물 지하) | **메커니즘 유형**: 핫스팟 추론 배치 (맵 재방문 순례)
- **메커니즘 상세**: 유품 상자의 3점 — **안경집 / 뼈 피리 악보 / 타다 만 장부 반쪽**. 각 방의 잔상 장면 속에서 그 물건이 "그날 밤 있던 자리"를 찾아 후보 핫스팟 3곳 중 정위치에 놓는다(물건 탭 → 핫스팟 탭). 근거는 잔상 자체: ① 안경집 → 저장고, 배급하는 손 곁의 장부 서랍 옆(눈먼 그녀는 장부 곁에 늘 안경을 두었다) ② 악보 → 온실 모이통 아래(새를 부르던 자리) ③ 장부 반쪽 → 지하 가마 앞(불에 타다 만 이유가 거기 있으므로).
- **UI 스케치**: 그날 밤 모드 팔레트(차가운 청색 + 잔상 실루엣). 물건을 놓으면 잔상이 잠시 선명해지는 보상 연출.
- **정답/클리어**: 3점 모두 정위치.
- **힌트1**: 잔상을 봐라. 물건은 주인의 손버릇이 두던 자리로 돌아가고 싶어 한다.
- **힌트2**: 안경은 글씨 곁에, 악보는 새 곁에, 장부는 — 불 곁에.
- **힌트3**: 장부 서랍 옆 / 모이통 아래 / 가마 앞 재 위.
- **정답 공개**: "세 물건이 제자리로 돌아가자, 잔상들이 잠시 숨을 쉰다. 반납이 아니라 — 애도다."
- **오답 처리**: 오배치 시 잔상이 조용히 고개를 젓는다(텍스트 없음). 무벌점.
- **아이템**: 소모 — 유품 3점(각 방에 안치). 8개 방 잔상 장면 카드가 수첩에 자동 수집 완료(→S31).

## S31 [퀴즈] 그날 밤 타임라인 — 8장면 배열 — 난이도 5
- **방**: 전 방(제출은 응접실) | **메커니즘 유형**: 탭 순서 배열 — **S02와 완전 동일 UI 재사용** (인수인계 메모 3번: 같은 손짓으로 정반대 이야기를 배열하게 하는 것이 체감 반전의 핵심)
- **메커니즘 상세**: 잔상 장면 카드 8장을 시간순으로 탭 배열. 화면 상단에 S02에서 플레이어가 맞춘 동화 삽화 6장 순서가 고정 표시되어, 아래의 진짜 8장면과 화면 분할 대비. **정답 순서**: ① 온실 — 모이 길을 따라 걸어오는 두 아이(겨울) ② 아이들의 방 — 간호와 몸무게 기록 ③ 부엌 — 빵 굽는 낮 ④ 다락 — 아이가 성에에 글씨를 긁는 밤 ⑤ 저장고 — 밤마다 식량을 받아 가는 마을의 손들 ⑥ 응접실 — 문이 부서지고 도끼가 벽을 찍는 밤 ⑦ 우물 지하 — 가마 앞, 걸쇠에 얹힌 작은 손 ⑧ 뒷방 — 이튿날 새벽, 소년의 첫 설탕 칠.
- **UI 스케치**: S02와 동일한 카드 그리드·뱃지·확정 버튼(단 8장, 청색 팔레트). 상단 띠에 S02 결과 6장 썸네일.
- **정답/클리어**: 8장면 정순 배열.
- **힌트1**: 계절과 빛을 봐라. 도착은 겨울, 습격은 밤, 설탕은 새벽이다.
- **힌트2**: 배급(저장고)은 오래 계속된 일 — 하지만 카드 속 장면은 '들킨 밤'에 가장 가깝다. 습격 직전에 놓아라.

- **오답 처리**: S02와 동일(틀린 장수만 표시). 3회 오답 시 카드의 시간 단서(달·촛불·새벽빛) 확대.
- **아이템**: 타임라인 완성 기록 → S32 활성.

## S32 [미션] 가마 앞 재연 — 걸쇠의 기억 — 난이도 4
- **방**: 우물 지하 | **메커니즘 유형**: 시퀀스 인터랙션 + 내재 선택지 (오답 없음)
