### PUZZLE 01 — 프리즘 레이저 조준 (관측실) [수정 포인트]
```
분류 태그:  PUZZLE 01 · 관측실
제목:       프리즘 레이저 조준
설명:       레이저 소스에서 발사된 빔을 거울 A, B, C의 각도를 조절하여
            망원경 렌즈(TARGET)에 맞추세요.
            거울 C는 빔 경로에 없을 수 있습니다.
힌트1:      A와 B 두 거울만 사용하면 됩니다.
힌트2:      A를 135°로, B도 135°로 맞춰보세요.
정답:       A=135°, B=135°
```

### PUZZLE 09 — 원자로 코어 균형 조절 (동력실) [수정 포인트]
```
분류 태그:  PUZZLE 09 · 동력실
제목:       원자로 코어 균형 조절
설명:       벽면 다이어그램: α코어 기준 출력 85MW, β코어 피크 120MW, 총 출력 300MW.
            세 코어의 합이 300MW가 되도록 슬라이더를 조절하세요.
힌트1:      α=85, β=120이면 γ는 자동으로 계산됩니다.
힌트2:      300-85-120 = 95. γ를 95에 맞추세요.
정답:       α=85, β=120, γ=95
```
