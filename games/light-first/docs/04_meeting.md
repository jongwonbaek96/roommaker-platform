## 10. 현재 스토리 문제점 (검토 의견)

> 수정 시 참고할 현재 구조의 약점들

1. **연구 일지에 단서 과다집중**
   - 문서 3(연구 일지)에 서버 코드, DNA 패턴, 동력실 순서까지 모두 기재됨
   - → 단서를 방별로 분산시키면 탐색의 재미가 올라감

2. **챕터 설명이 정보 전달 위주**
   - 현재: "BSL-4 등급의 격리 구역. 외계 생물 샘플이 보관되어 있다."
   - → 감정/분위기/긴장감을 살리는 문장으로 개선 가능

3. **Dr. 한이준의 캐릭터가 납작함**
   - 왜 시간 이동을 선택했는지, 어떤 사람인지 묘사 없음
   - → 문서들에 개인적인 감정 묘사 추가 가능

4. **외계 생물체(α-라이라) 설정이 미활용**
   - DNA 퍼즐에만 등장하고 스토리와 연결이 없음
   - → 신호의 발원지와 연결하거나, 삭제하거나 결정 필요

5. **biocode(4321) 단서가 불명확**
   - 격리실 기록판이 SVG 씬에 어떻게 그려져 있는지 확인 필요
   - → 플레이 시 단서가 명확히 보이는지 체크

6. **인트로에 이미지 없음**
   - 텍스트만으로 분위기 전달 한계
   - → 달 기지 이미지, Dr. 한이준 프로필 이미지 추가 검토
