> 게임 id: `light-first` · SF 스릴러 · LUMOS-7 달 기지
> ⚠️ **역추출 문서** — 원본 기획서에 연출 항목이 없어 게임 HTML(`games/light-first.html`)에서 실측해 작성했다.

---

## 1. 이미지 자산 — **없음**

이 게임은 **외부 이미지를 전혀 쓰지 않는다.** 모든 화면은 CSS + 이모지 + 유니코드 기호로 그린다.
따라서 "이미지 프롬프트" 항목이 존재하지 않으며, 비주얼 수정은 곧 **CSS 수정**이다.

| 구분 | 실제 구현 |
|---|---|
| 배경 | 단색(`--bg`) + 그리드/스캔라인 CSS |
| 오브젝트 | 이모지 · 박스 · 테두리 |
| 문서(수집물) | 모노스페이스 텍스트 패널 |
| 퍼즐 장치 | CSS/DOM으로 직접 그림 (레이저·거울·배선도 등) |

> **주의**: 이미지를 넣고 싶다면 `games/assets/light-first/` 를 새로 파고 HTML에 `<img>`를 추가해야 한다.
> `/games/assets/*` 는 게이트에서 제외돼 있어 정적으로 서빙된다.

## 2. 컬러 팔레트 (게임 HTML의 CSS 변수 — 여기가 정본)

| 변수 | 값 | 용도 |
|---|---|---|
| `--bg` | `#050a14` | 기지 배경(심우주 남색) |
| `--panel` | `#0a1628` | 패널·모달 |
| `--border` | `#1a3a5c` | 테두리 |
| `--text` | `#c8e0f0` | 본문 |
| `--dim` | `#4a6a8a` | 보조 텍스트 |
| `--accent` | `#00c8ff` | 강조 · 홀로그램 시안 (registry accent와 동일) |
| `--success` | `#00ff88` | 정답 · 해제 |
| `--danger` | `#ff3355` | 경보 · 오답 |
| `--gold` | `#ffd700` | 수집 문서 · 핵심 단서 |

퍼즐별 광원 색(`--c`)으로 시안 `#00c8ff` / 그린 `#00ff88` / 오렌지 `#ff6b35` / 골드 `#ffd700` 를 돌려 쓴다.
**색광 합성 퍼즐(PUZZLE 03)** 이 이 팔레트에 직접 의존하므로, 색을 바꾸면 퍼즐 정답도 같이 확인할 것.

## 3. 타이포그래피

- 본문: `'Malgun Gothic','맑은 고딕', sans-serif`
- 터미널 · 로그 · 코드 패널: `'Courier New', monospace` — **기지 시스템의 목소리**는 전부 모노스페이스로 통일한다.

## 4. 연출 (CSS 애니메이션)

| keyframes | 쓰이는 곳 |
|---|---|
| `pulse-border` | 상호작용 가능한 오브젝트 테두리 맥동 |
| `pulse-dim` | 잠긴/비활성 오브젝트 |
| `flash-success` | 퍼즐 해결 순간 섬광 |
| `notif-in` / `notif-out` | 토스트 알림 등장·퇴장 |

연출 톤: **정전된 기지에 홀로 남은 감각.** 급격한 컷보다 느린 페이드·맥동을 쓴다.
챕터 오버레이(→ `01_story.md` §3)가 방 전환의 유일한 "컷"이다.
