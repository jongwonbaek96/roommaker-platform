## 1. 이미지 자산 — **없음**

## 2. 팔레트 (실측 — 시리즈에서 유일하게 **밝은 배경**)

| 변수 | 값 | 용도 |
|---|---|---|
| `--cream` | `#fff8f5` | **배경** (registry bg — 다른 게임은 전부 검정 계열) |
| `--parch` | `#fdf6e3` | 편지지 |
| `--parch2` | `#f5e6c0` | 낡은 편지지 |
| `--ink` | `#4a3520` | 본문 (잉크색) |
| `--rose` | `#e8637a` | 강조 (registry accent) |
| `--pink` | `#f2a7bb` · `--soft` `#fde8ee` | 회상 장면 |

**색 설계 의도**: 편지지와 잉크. 이 게임은 **읽는 게임**이므로 종이의 질감이 곧 UI다.
회상 장면(`MEMORY`)은 `--pink`/`--soft` 쪽으로 살짝 물들이고, 하람의 목소리(`HARAM'S ...`)는 `--parch2`(낡은 편지지)로 구분한다.

## 5. BGM — **있다** (시리즈에서 sugar-melts와 함께 유이하게)

Web Audio 오실레이터로 **외부 파일 없이** 멜로디를 연주한다. 토글 가능(`bgmOn`).

```js
const BGM_NOTES = [523.25, 659.25, 783.99, 880, 659.25, 523.25, 440, 523.25];
//                  도5     미5     솔5    라5   미5     도5    라4   도5
```

- **C major 계열의 단순한 8음 루프.** 도–미–솔–라–미–도–라–도.
- 오실레이터 + 게인 엔벨로프(`playNote`)로 각 음을 `d*0.88` 길이만큼 울린다.
- **왜 이 구조가 좋은가**: 외부 파일 0개 = 로딩 없음 = 단일파일 자체완결 유지. **다른 게임에 BGM을 넣을 때 이 구현을 그대로 재사용할 것.**

### 수정 시 주의

- 음을 바꾸려면 `BGM_NOTES` 주파수 배열만 고치면 된다.
- 이 루프는 **끝없이 반복**된다. 감성 게임이므로 **단조롭고 조용한 편이 낫다.** 화려하게 만들지 말 것.
- 엔딩 3종에 각각 다른 루프를 주면 분기감이 훨씬 살아난다. (현재는 전부 동일 — **개선 후보**)
