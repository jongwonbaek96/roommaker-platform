# 색깔없는세상 🟢 초록 — 이미지 · BGM · 연출

> 게임 id: `colorless-green` · 엔진 `colorless-core.js`
> ⚠️ v4 미반영 — 현재 게임은 `colorless-core.js`(구버전) 이다.

---

## 1. 채색 시스템 (시리즈 핵심 연출)

세상은 **흑백**이다. 문제를 풀 때마다 진행도가 오르고, 그에 비례해 **캐릭터 아바타(SVG)와 UI가 점점 색으로 물든다.**
30번을 풀면 **채색 100%** → 엔딩 리빌 몽타주 → PRISM 조각.

- 구현: `colorless-core*.js` 의 채색 로직 + `colorless-style.css`
- **한 곳을 고치면 5개 에피소드 전부 바뀐다.** (공유 엔진)

## 3. 폰트

- 일기·손글씨: **Gaegu** (Google Fonts)
- UI·본문: **IBM Plex Sans KR**
- 두 폰트 모두 `colorless-*.html` 의 `<head>` 에서 CDN 로드. **오프라인에서는 폰트가 깨진다.**

## 4. 이미지 자산

`window.GAME.assets` = `undefined` — 그러나 **현재 게임은 이미지를 한 장도 참조하지 않는다** (실측 0장).
모든 화면이 CSS + 이모지로만 그려진다.

### v4 기준 필요 자산 (미제작)

v4는 그림일기 삽화를 전제로 한다. 이 에피소드를 v4로 재빌드하려면:

| 파일 | 내용 |
|---|---|
| `assets/green/d1.webp` ~ `d5.webp` | 5장 각각의 그림일기 삽화 (크레파스 화풍, **해맑음 100%**) |
| `assets/green/s_*.webp` | 주요 미션 배경 (선택) |
| 리빌 이미지 1장 | 엔딩 몽타주 마지막 컷 — `01_story.md` §3 참고 |

> 이미지 톤 절대 규칙: **엔딩 전 이미지에는 진실의 냄새가 단 하나도 없어야 한다.**
> 빨강(`colorless-red`)의 자산 명명 규칙을 그대로 따를 것.

## 5. BGM · 사운드

**없음.** 5개 에피소드 전부 오디오를 쓰지 않는다 (실측 0건).
넣는다면 공용 엔진 `colorless-core*.js` 에 한 번만 추가하면 5편 전부 반영된다.

- 권장: 흑백 구간엔 뮤트/저역만, **채색이 진행될수록 악기가 하나씩 들어오는** 구조. (채색 시스템과 같은 축)
- 엔딩 리빌: 몽타주 5컷 동안 음이 쌓이다 진실 문장에서 **뚝 끊기는** 연출.
